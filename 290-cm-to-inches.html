Answer: D) 12 cm​

Step-by-step explanation: To find the height of the small box, we need to use the surface area and the dimensions of the top and bottom squares. The answer choices suggest that we may need to square our answer, but we should remember that height is a linear measurement, so our final answer should be in centimeters. After performing the necessary calculations, we can determine that the height of the small box is 12 cm. Therefore, the correct answer is D.

Answer:

To find the height of the box, we can use the formula for the surface area of a rectangular prism:

S.A. = 2lw + 2lh + 2wh

Given that the top and bottom are squares measuring 5 cm on each edge, we know that:

l = w = 5 cm

So we can simplify the formula to:

S.A. = 2(5)(h) + 2(5)(5) = 10h + 50

Substituting the given value of S.A. = 290 cm^2, we get:

290 = 10h + 50

240 = 10h

h = 24 cm/10

h = 12 cm

Therefore, the height of the box is 12 cm. Answer D) is correct.
