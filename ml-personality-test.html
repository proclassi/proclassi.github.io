1. According to this multistage fitness test reference chart, what would the maximum oxygen intake be for a person who failed to complete the seventh shuttle run in the fifth level?a.26.8 ml/kg/minb.28.3 ml/kg/minc.30.2 ml/kg/min


C. 30.2 ml/kg/min

The multistage fitness test is also known as the beep test or bleep test is a 20m shuttle run test. It requires an athlete to continuous the 20m shuttle runs. The athlete needs to reach the end of the 20m grid before the beep sounds. The time between the beep sound decreases every minute, therefore, the athlete must increase his/her speed.

This test measures the aerobic power of an individual in a sport such as Soccer, Football, Rugby, Hockey and others. 

To Calculate
VO^2
VO ^2 max = - age + speed 

Speed is the final 20m shuttle run before the test withdrawal.


Total Distance
 It is measured by recording the total number of shuttles of an athlete multiply by 20.

Level Achieved
It is the athlete level reach. The number of shuttles.
