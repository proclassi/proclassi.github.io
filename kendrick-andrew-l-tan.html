Problem:AYLA which is right-angled at A. If y = 15 and a = 17. Determine the six trigonometric ratios for L

Solution:

Use pythagorean theorem

a² = y² + I²

17² = 15² + I²

289 = 225 + I²

I² = 289 - 225

I² = 64

I = √64

I = 8

Sine L = opposite side / hypotenuse

Cosine L = adjacent side / hypotenuse

Tangent L = Opposite side / adjacent side

Cosecant L = hypotenuse / opposite side

Secant L = hypotenuse / adjacent side

Cotangent L = adjacent side / opposite side

sin L = 8/17

cos L = 15/17

tan L = 8/15

csc L = 17/8

sec L = 17/15

cot L = 15/8
