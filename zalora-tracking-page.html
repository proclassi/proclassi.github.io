27. The circumference of a circular track is 154cm.Find the diameter of the track


Circumstances Formula:

C = 2πr

where:
C = 154cm
π = 22/7
r = ?

so,

C = 2πr
154 = 2(22/7)( r )
154 = 44/7 r
154 / 44/7 = 44/7 / 44/7 r
539/22 = r

the radius is 539/22

Diameter Formula:

D = 2r
D = 2(539/22)
D = 49 cm
