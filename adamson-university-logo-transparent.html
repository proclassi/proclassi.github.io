22. wood isn't transparent or transparent or transparent​


Answer:

Wood is not transparent, it is opaque. Light cannot pass through it so it is opaque.

Additional Info:

Transparent - Light can easily pass through in a transparent materials.

Translucent - Light can pass through in a translucent object but not all of the light can pass through it.

Opaque - Light cannot pass through an opaque materials.
